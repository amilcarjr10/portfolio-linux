# Scripts de Automação de Segurança (Hardening e Monitoramento)

Esta pasta contém o projeto de destaque do portfólio, desenvolvido para simular a rotina de um Analista de Segurança/Operações:

- **security_monitor.sh:** Script em Bash que automatiza o checklist diário de segurança (falhas de login, mapeamento de portas, status do firewall e versão do Kernel).
