# Auditor de Segurança de Contas de Usuário (Python)

## Visão Geral

Este projeto demonstra as capacidades de scripting em Python para auditoria de segurança e conformidade em TI. O script lê o arquivo local `/etc/passwd` para identificar e sinalizar contas com privilégios elevados (UID 0 - Root) ou com shells potencialmente inseguros/desabilitados.

## Competências Demonstradas

- Programação Python (Manipulação de I/O de Arquivos, Strings e Funções)
- Fundamentos de Auditoria de Segurança
- Conhecimento de Sistemas Linux (Estrutura do arquivo /etc/passwd)
